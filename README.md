# Python Assignment — Part 4: Data Visualization & Machine Learning

## Overview
This project analyses a student performance dataset, produces meaningful visualizations, and builds a machine learning model to predict whether a student will pass or fail.

## Tasks Covered
- Task 1: Data Exploration with Pandas
- Task 2: Data Visualization with Matplotlib (5 plots)
- Task 3: Data Visualization with Seaborn (2 plots)
- Task 4: Machine Learning with scikit-learn

## How to Run
Open part4_visualization_ml.ipynb in Google Colab and run all cells.
